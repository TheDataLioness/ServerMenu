# ServerMenu


Make sure you have the FormAPI plugin on your server

Edit Servers in config.yml (config.yml will be created on plugin startup)


## Usage:
### Commands:
      /servermenu
## Call from other plugin:
  Call the serverMenu() function from the main class.
